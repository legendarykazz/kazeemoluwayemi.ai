# Portfolio Website

A personal portfolio website design for an AI Automation Engineer & Web Designer.

## Features
- **Modern Design**: Deep Blue / Sky Blue aesthetic with Glassmorphism.
- **Dark/Light Mode**: Fully themeable with automatic system detection.
- **Responsive**: Mobile-first layout.
- **Animations**: Performance-focused scroll reveal effects.
- **Pages**: Home (One-page style) + Dedicated About Me page.

## How to Run

### Option 1: Direct Open
Since this uses Vanilla JS and CSS, you can simply open `index.html` in your web browser.
*Note: Some modules might encounter CORS issues depending on the browser security settings if not served via a server.*

### Option 2: Live Server (Recommended)
If you have VS Code, install the "Live Server" extension and click "Go Live" at the bottom right.

### Option 3: Vite (If Node.js is installed)
1. Run `npm install`
2. Run `npm run dev`
3. Open the localhost URL shown in the terminal.

## Customization
- **Colors**: Edit `src/style.css` variables in `:root`.
- **Content**: Update text in `index.html` and `about.html`.
